### Getting Started

```git clone https://github.com/dubrovsky-kiril/scout-rfp.git```


### Techology stack
- React
- Redux


### Prerequisites

- Node (https://nodejs.org/en/)
- Yarn (https://yarnpkg.com/lang/en/)


### Installing dependencies

```
yarn install
```


## Running linting

```
yarn lint
```


## Running tests

```
yarn test
```
